# EthernetESPThreadBorderRouter
Forked from https://github.com/arduino-libraries/Ethernet. Provides support for [ESP Thread Border Router](https://docs.espressif.com/projects/esp-thread-br/en/latest/hardware_platforms.html#ethernet-based-thread-border-router).

## Usage
Add `BOARD_ESP_THREAD_BORDER_ROUTER` to your `build_flags`:
```ini
[env]
build_flags =
;   ...
    -D BOARD_ESP_THREAD_BORDER_ROUTER
```
